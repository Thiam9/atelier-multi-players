Bienvenue dans calcule mentale Game

Ce jeu de maths multijoueurs est un projet que j'ai réalisé pour ma formation Full stack JS. Le jeu est codé avec Socket io, node Js ,express js, HTML et Tailwind css. Plusieurs utilisateurs peuvent s'affronter en même temps. Le but du jeu consiste à répondre aux questions le plus rapidement possible, le plus rapide aura le plus de points. Bonne chance!
